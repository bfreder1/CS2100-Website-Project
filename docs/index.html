Welcome to the Grecian Realms Wiki

The Grecian Realms is the setting for a D&D campaign originally penned by Bryson Frederick in Spring of 2019. This wiki is a collaborative effort to document the world and its inhabitants. The campaign is set in a world inspired by ancient Greece, with a pantheon of gods, mythical creatures, and epic heroes. More fantasical elements are brought from the Forgotten Realms and other D&D settings, hence the name "Grecian Realms".

More about the history of the campaign and its author can be found in the About page. Contributions are welcome and encouraged, to help expand the world and make it more immersive for all players. If you would like to contribute, please see the Contribute page for more information. Thank you for visiting the Grecian Realms Wiki!
